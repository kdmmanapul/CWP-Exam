# CounterWebProductions Time Tracker Exam

## How to use

Download the example [or clone the repo](https://github.com/kdmmanapul/CWP-Exam.git):

```bash
git clone https://github.com/kdmmanapul/CWP-Exam.git
cd CWP-Exam
```

Install it and run:

```bash
npm install
npm start
```

## Libraries and Frameworks Used
ReactJS
Material UI
MomentJS
Styled-Components
React Router
